### Node path

1. path.join (Absolute path)

```java
path.join('/user', '/ryan', 'programs');    // returns "/user/ryan/programs"
```

2. path.resolve (Relative path)

```java
path.resolve('/user', '/ryan', 'programs');    // returns "/ryan/programs"
```
